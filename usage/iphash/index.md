## 场景

根据源 IP 进行负载均衡，在 istio 中如何配置呢 ？

## 用法

配置 `DestinationRule`，指定 `useSourceIp` 负载均衡策略:


```yaml
apiVersion: networking.istio.io/v1beta1
kind: DestinationRule
metadata:
  name: bookinfo-ratings
spec:
  host: ratings.prod.svc.cluster.local
  trafficPolicy:
    loadBalancer:
      consistentHash:
        useSourceIp: true
```
